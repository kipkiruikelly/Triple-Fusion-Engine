Data Sources

Every provider that powers BullLogic and exactly what each supplies.
Yahoo Financeprimary prices

Live quotes, historical daily and intraday candles for stocks, ETFs, crypto, forex, commodities, and indices. Used for model training, live predictions, charts, and the price sidebar. Data may be delayed by up to 15 minutes for some exchanges.

Pyth Network (Hermes)verification + failover

An oracle network aggregating prices from major exchanges and trading firms, delivered with a confidence interval. We cross-check displayed prices against Pyth and show a Verified badge when sources agree. If Yahoo is unavailable, Pyth prices take over with a clear source label. Equity feeds pause outside US market hours by design.

Alpaca Marketstraining data

High-resolution 1-minute historical bars used to train the short-timeframe models. Not used for live display.

Microsoft Azure Blob Storageinfrastructure

Encrypted storage for trained model files. No personal data is stored there.

Safaricom Daraja and Stripepayments

Payment processing only. See the Privacy Policy for what payment data we retain.

Where sources disagree beyond tolerance we show both values rather than choosing silently. When all sources fail you see an honest "last known price" banner with the timestamp, never a fabricated number.
