How It Works
📊

ICT Feature Engineering

Every bar is annotated with Order Blocks, Fair Value Gaps, Change of Character, OTE zones, PD arrays, kill zones, and IPDA liquidity levels — the same concepts used by institutional traders.

View Model AUC Scores →
🎯

Multi-Timeframe Confluence

Higher-timeframe bias (1d, 4h) is injected into lower-timeframe models (1h, 15m) via forward-fill, so every signal inherits the structural context of the full market structure.

Run a Prediction →
⚖

Ensemble ModelsPRO

Three classifiers vote on each signal: Logistic Regression for linear structure, Random Forest for non-linear patterns, and XGBoost with 5-fold CV for robustness. Final confidence is ensemble-weighted.

Run a Backtest →
🚀

Automated Retraining

A systemd timer fires every Sunday at 2 AM, runs the full training pipeline with the latest Alpaca data, saves model metrics, and auto-uploads artifacts to Azure Blob Storage.
